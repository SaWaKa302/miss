# 🗾 Anime Seichi Map

An interactive mobile-first map for international anime fans visiting Japan — find real-world filming locations, plan your pilgrimage route, and explore Japan through the anime you love.

**🌐 Live site:** https://sawaka302.github.io/Anime-Nippon/

---

## What is this?

*Seichi junrei* (聖地巡礼) — anime pilgrimage — is the practice of visiting real-world locations that appeared in anime. This app makes that experience accessible to international visitors with English descriptions, Google Maps navigation, and smart route planning.

## Features

- 📍 **127+ spots** across 10 anime series
- 🗺️ **Pilgrimage mode** — set your own visit order and navigate spot to spot
- ✨ **Pilgrimage Guide** — answer 3 questions and get a recommended route based on your schedule and preferred time of day
- 🕐 **Best time badges** — each spot shows the ideal time to visit
- 📝 **Personal notes** — save your own memos per spot
- 📬 **Feedback** — rate the app directly from the pilgrimage completion screen

## How to use

1. Open the site on your phone browser
2. Tap **Select Anime** to choose a series
3. Choose **Pilgrimage Guide** (recommended route) or **Plan yourself** (manual order)
4. Follow the route — tap any pin for details, access info, and Google Maps navigation
5. Tap **Arrived!** at each spot to mark it visited

## Tech stack

Single HTML file — no framework, no build step, no backend.

- [Leaflet.js](https://leafletjs.com/) — map rendering
- [CartoDB Positron](https://carto.com/basemaps/) — English map tiles
- [Tabler Icons](https://tabler.io/icons) — UI icons
- Hosted on GitHub Pages
